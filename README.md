# Super-Store-Sales-Analysis-Dashboard
Built a Super Store Sales Analysis Dashboard by cleaning and transforming sales data. Created KPIs, analyzed regions, categories, and profit trends, and applied Power BI forecasting to predict future sales. Insights help optimize inventory, improve discounts, and support data-driven business decisions.

📊 Super Store Sales Analysis Dashboard – Project Showcase

Excited to share my latest **Super Store Sales Analysis Dashboard** where I analyzed sales data to uncover actionable business insights and support data-driven decision-making.

🔍 Challenges Faced
• Cleaning inconsistent and messy data
• Handling missing values and duplicates
• Creating meaningful KPIs from raw data
• Designing an interactive and easy-to-use dashboard

📥 Data Collection & Preparation
• Collected Super Store sales dataset
• Cleaned and transformed data using Excel/SQL
• Structured data into region, category, segment, and product level
• Prepared dataset for visualization and analysis

📊 Dashboard Features
• Sales & Profit Overview
• Region-wise performance analysis
• Category & Sub-category insights
• Top-performing products
• Interactive filters for dynamic exploration
• Forecasting for future sales trends

💡 Key Insights
• West region generated the highest overall sales
• Technology category contributed the most profit
• Some sub-categories showed high sales but low profit margins
• Office Supplies had consistent but moderate growth
• Discounts heavily impacted profitability in certain products
• Seasonal spikes observed during year-end months

📈 Forecasting (Power BI)
• Applied Power BI forecasting to predict future sales
• Identified expected growth trends for upcoming months
• Helped estimate future demand and revenue
• Improved planning for inventory and sales strategy

🚀 Project Outcomes
• Identified high-performing regions and categories
• Detected low-profit products affecting margins
• Highlighted sales trends across time
• Predicted future sales using forecasting

🌍 Real-World Impact
• Helps businesses focus on profitable products
• Improves inventory and supply planning
• Supports regional sales strategy
• Enables smarter discount decisions
• Assists in future demand planning using forecasting

🛠️ Tools Used: Power BI / Tableau / Excel / SQL
